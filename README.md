## Jenkins Init

-  checkout the repo
-  create a folder named `jenkins_home`
- `docker-compose build`
- `docker-compose -d`
- add a line to /etc/hosts 

```bash
127.0.0.1 jenkins.local
```

- open chrome and visit [[http://jenkins.local:8080]]
